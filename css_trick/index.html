Overflow Property

The overflow property controls what happens to content that breaks outside of its bounds. For example, if you've explicitly set a div to be 200px wide, but contains an image that is 300px wide, that image will stick out of the div and be visible by default. Whereas if you set the overflow value to hidden, the image will cut off at 200px.
